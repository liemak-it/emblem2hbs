emblem2hbs
==========

Convert Emblem Templates to Handlebars

```
git clone git@github.com:patientslikeme/emblem2hbs.git
cd emblem2hbs
npm install -g
```

...somewhere else...

```
emblem2hbs somefile.emblem
```
